z-mongo-admin
===============

ZMA (ZMA Meteor Admin) is a meteorite package for Mongo Admin for Meteor.

![zma](http://www.goodnnatural.com/pub/products/12151_T1_W275_H.jpeg)

Install
----------
```
mrt install spin
mrt install router
mrt install z-mongo-admin
```
